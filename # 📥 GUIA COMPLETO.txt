# 📥 GUIA COMPLETO - Como Baixar Seu Site

## 🎯 MÉTODO MAIS FÁCIL - Arquivo HTML Único

### PASSO 1: Abrir o Editor de Texto
- **Windows:** Abra o "Bloco de Notas" (Notepad)
  - Aperte a tecla Windows
  - Digite "Bloco de Notas"
  - Clique para abrir

- **Mac:** Abra o "TextEdit"
  - Cmd + Espaço
  - Digite "TextEdit"
  - Clique para abrir

### PASSO 2: Copiar o Código
1. Volte para a conversa aqui no Claude
2. Procure o artefato chamado **"Site Agência Inteligente - HTML Único"**
3. No canto superior direito do artefato, clique nos 3 pontinhos (⋮)
4. Clique em "Copy code" ou "Copiar código"
5. Todo o código será copiado!

### PASSO 3: Colar no Editor
1. Volte para o Bloco de Notas/TextEdit
2. Cole o código (Ctrl+V no Windows ou Cmd+V no Mac)
3. Você verá todo o código HTML aparecer

### PASSO 4: Salvar o Arquivo
**IMPORTANTE: Siga exatamente estas instruções!**

#### No Windows (Bloco de Notas):
1. Clique em "Arquivo" → "Salvar Como"
2. Em "Nome do arquivo", digite: `agencia-inteligente.html`
3. Em "Tipo", selecione: **"Todos os arquivos (*.*)"**
4. Escolha onde salvar (pode ser na Área de Trabalho)
5. Clique em "Salvar"

#### No Mac (TextEdit):
1. Clique em "Arquivo" → "Salvar"
2. Digite o nome: `agencia-inteligente.html`
3. Escolha onde salvar (pode ser na Área de Trabalho)
4. Clique em "Salvar"

### PASSO 5: Abrir o Site
1. Vá até onde você salvou o arquivo
2. Dê **duplo clique** no arquivo `agencia-inteligente.html`
3. O site abrirá automaticamente no seu navegador! 🎉

## 🔧 SE DER ERRO:

### Erro: "Arquivo não abre no navegador"
**Solução:**
1. Clique com botão direito no arquivo
2. Escolha "Abrir com"
3. Selecione seu navegador (Chrome, Firefox, Edge, Safari)

### Erro: "Código aparece como texto"
**Solução:**
- Você não salvou com a extensão .html correta
- Refaça o PASSO 4, garantindo que o nome termine em .html
- No Windows, certifique-se de selecionar "Todos os arquivos"

### Erro: "Site não funciona corretamente"
**Solução:**
- Você precisa estar conectado à internet
- O site usa bibliotecas online (React, Tailwind)
- Certifique-se de ter conexão com a internet
